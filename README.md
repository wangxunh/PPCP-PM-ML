# PPCP-PM-ML

Machine learning models for predicting the removal efficiency of pharmaceuticals and personal care products (PPCPs) by photocatalytic membranes.

## Overview

This repository contains the data and Python codes used to develop machine learning models for predicting PPCP removal efficiency by photocatalytic membranes. The study evaluates multiple machine learning algorithms and molecular representations to analyze the effects of PPCP properties, photocatalyst characteristics, membrane materials, and operational conditions on removal performance.
